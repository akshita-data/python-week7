# 📊 Week 7: Statistical Business Analysis

## 🎯 Project Overview
This project focuses on applying **statistical techniques** to analyze business data and extract meaningful insights.  
The goal is to move beyond basic analysis and use **statistics to support decision-making**.

---

## 📁 Dataset
The project uses business datasets such as:
- `sales_data.csv` – Contains sales transactions (price, quantity, revenue)
- `customer_churn.csv` – Contains customer behavior data

---

## 🧠 Concepts Covered

### ✔ Descriptive Statistics
- Mean
- Median
- Mode
- Standard Deviation

### ✔ Data Distribution
- Histogram
- Density plots
- Normal distribution understanding

### ✔ Correlation Analysis
- Pearson correlation
- Relationship between variables

### ✔ Hypothesis Testing
- T-test (comparison between groups)
- P-value interpretation

### ✔ Confidence Intervals
- 95% confidence interval
- Margin of error

### ✔ Regression Analysis
- Linear relationship between variables
- R-squared evaluation
